# 宠物伙伴指南

Snow CLI 内置了一个由 `/buddy` 指令管理的本地终端宠物伙伴。宠物会保存在本地，可以显示在终端界面中，并在你工作时通过本地气泡做出简短反应。

返回：[目录](./0.目录.md)

---

## 功能概览

宠物伙伴是一个轻量级的本地编码陪伴功能。

- **本地状态**：宠物数据保存在用户本地 Snow 配置目录中。
- **终端界面**：启用时，宠物可以显示在输入区域附近。
- **互动指令**：支持孵化、摸摸、改名、对话、静音、取消静音、查看状态和重置。
- **随机特征**：孵化时会随机生成稀有度、眼睛、帽子、闪光状态和属性。
- **可选宠物类型**：孵化时可以指定宠物类型；不指定时默认随机。

---

## 指令总览

```bash
/buddy
/buddy status
/buddy hatch [名字] [--species=类型] [--list-species] [--personality=文本]
/buddy pet
/buddy rename <名字>
/buddy say <消息>
/buddy profile [list|current|default|reset|<配置名>]
/buddy mute
/buddy unmute
/buddy reset
```

不带参数的 `/buddy` 等同于 `/buddy status`。

---

## 孵化宠物

使用 `/buddy hatch` 创建你的宠物伙伴。

```bash
/buddy hatch
/buddy hatch Mochi
```

不提供名字时，Snow CLI 会随机选择一个默认名字。不提供宠物类型时，类型会随机生成。

### 指定宠物类型

可以通过 `--species=` 指定宠物类型：

```bash
/buddy hatch Mochi --species=cat
/buddy hatch Orbit --species=dragon
```

这个选项只固定宠物类型。帽子仍然随机，稀有度、眼睛、闪光状态和属性也仍按原逻辑随机生成。

### 查看可选类型列表

可以使用以下任一指令打印所有支持的宠物类型：

```bash
/buddy hatch --list-species
/buddy hatch --species=list
```

当前可选类型：

```text
duck, goose, blob, chicken, basketball, cat, dragon, octopus, owl, penguin, turtle, snail, ghost, axolotl, capybara, cactus, robot, rabbit, mushroom, chonk, fox, panda, raccoon, unicorn, whale, hamster, teapot, rocket, laptop, moon, cloud, lantern, treasure, book, star, coffee, snowman
```

### 添加性格描述

可以通过 `--personality=` 提供性格描述：

```bash
/buddy hatch Pip --species=fox --personality=curious, loyal, and fond of tests
```

性格文本可以包含空格。`--personality=` 之后的所有内容都会被当作性格描述。

## 与宠物互动

### 查看状态

```bash
/buddy status
```

状态输出会展示宠物的名字、类型、稀有度、性格、帽子、眼睛、静音状态、AI 配置、孵化时间和属性。

### 摸摸宠物

```bash
/buddy pet
```

该指令会触发一次短暂的本地 UI 反应。如果配置了宠物回复模型，宠物也可能在气泡中回复。

### 与宠物说话

```bash
/buddy say hello
/buddy say how are the tests looking?
```

宠物回复与主助手回复相互独立。宠物会作为本地伙伴在 UI 气泡中回答。

### 选择宠物 AI 配置

宠物 AI 回复默认使用当前 Snow Profile。你可以将宠物固定到指定 Profile、查看可用 Profile、查看当前选择，或重置为继续跟随当前 Snow Profile。

```bash
/buddy profile list
/buddy profile current
/buddy profile default
/buddy profile reset
/buddy profile work
```

选择的宠物 AI Profile 会持久化到本地 Snow 配置目录下的全局宠物状态文件中。未设置宠物专用 Profile 时，切换主 Snow Profile 也会改变宠物 AI 回复使用的配置。

### 给宠物改名

```bash
/buddy rename Noodle
```

改名只会修改宠物名字。宠物类型、稀有度、帽子、眼睛、性格、孵化时间和属性都会保持不变。

## 静音、取消静音与重置

### 静音

```bash
/buddy mute
```

静音后会隐藏 UI 宠物，并从提示词上下文中移除宠物信息，直到取消静音。

### 取消静音

```bash
/buddy unmute
```

取消静音后，宠物会重新显示，并允许使用宠物上下文。

### 重置

```bash
/buddy reset
```

重置会移除当前宠物。之后可以再次使用 `/buddy hatch` 创建新宠物。新的孵化会重新随机生成伙伴；如果还想指定某个类型，需要再次传入 `--species=`。

## 使用示例

```bash
# 随机孵化一个随机名字的宠物
/buddy hatch

# 使用自定义名字，宠物类型随机
/buddy hatch Mochi

# 孵化一只名为 Mochi 的 cat，帽子和其他特征仍随机
/buddy hatch Mochi --species=cat

# 查看所有支持的宠物类型
/buddy hatch --list-species

# 孵化一只带自定义性格的 fox
/buddy hatch Pip --species=fox --personality=curious, loyal, and fond of tests

# 摸摸宠物并与宠物说话
/buddy pet
/buddy say nice work today

# 将宠物 AI 回复固定到某个 Profile
/buddy profile list
/buddy profile work

# 临时隐藏宠物
/buddy mute
/buddy unmute
```

---

## 注意事项

- 同一时间只能存在一个宠物。孵化新宠物前请先使用 `/buddy reset`。
- 宠物类型名称为小写，并且必须与可选列表中的值一致。
- 如果提供了未知宠物类型，Snow CLI 会输出可选类型列表。
- 指定宠物类型不会指定帽子。帽子会按设计继续随机生成。
- `/buddy profile reset` 和 `/buddy profile default` 会清除宠物专用 AI Profile，使宠物重新跟随当前 Snow Profile。
